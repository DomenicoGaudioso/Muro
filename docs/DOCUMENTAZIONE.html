Muro di Sostegno

Web app Streamlit per verifiche geotecniche e strutturali di muri di sostegno a mensola, con analisi statica, sismica pseudo-statica, falda e tiranti.

Funzioni

Calcolo spinte attive e passive, verifiche a ribaltamento, scorrimento, portanza e sollecitazioni allo spiccato.

Output

Grafici Plotly, tabelle riepilogative, export input JSON e relazioni Word/PDF.

Normativa

Riferimenti operativi a NTC 2018, EC7-1 ed EC8-5 per il modello di verifica implementato.

Avvio rapido

Installa le dipendenze con pip install -r requirements.txt e avvia l'app con streamlit run app.py.

Il file app.py e l'entrypoint ufficiale del progetto.
